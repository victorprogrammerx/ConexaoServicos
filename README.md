📖 Sobre o Projeto

O Conexão de Serviços é um aplicativo desenvolvido como projeto acadêmico com o objetivo de conectar pessoas que precisam contratar serviços a profissionais que desejam oferecer seus trabalhos.

A plataforma permite a publicação, busca e contratação de serviços de diversas categorias, promovendo praticidade, rapidez e oportunidades para prestadores de serviços locais.

🎯 Objetivo

Criar uma solução digital que facilite a conexão entre clientes e profissionais autônomos, oferecendo uma experiência simples, intuitiva e acessível.

🚀 Funcionalidades do Protótipo
Login de usuários
Cadastro de usuários
Pesquisa de serviços
Visualização de serviços disponíveis
Detalhes do serviço
Candidatura para execução do serviço
Sistema de chat entre cliente e profissional
Cadastro de novos serviços
Perfil do usuário
Avaliações e reputação dos profissionais
